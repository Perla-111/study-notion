# <img width="21" height="25"  alt="image" src="https://github.com/user-attachments/assets/427864a4-335c-4c24-8915-235e655ef94f" /> <span align="top">StudyNotion</span>

A web application that allows users to explore a wide range of software development courses, view detailed content, and purchase them online. The platform helps learners upskill through structured, on-demand programming and development tutorials.

## 🛠 Tech Stack used

- `React`
- `Javascript`
- `Tailwind CSS`
- `Nodejs`
- `Express`
- `MongoDB`

## 🚀 Features
It provides features like ,
- course categorization and search for easy access
- user authentication, and secure payments
- watch course and track the course progress
- see ratings and reviews of courses

## 🚦 Running the Project

1. Clone the repository
2. Go to backend folder & install dependencies: `npm install`
3. Go to frontend folder & install dependencies: `npm install`
4. Run development server for each folders : `npm run dev`
5. Open `http://localhost:3000` in your browser
